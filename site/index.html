Finite APE Machine

Infinite monkeys produce noise. Finite APEs produce software.

v0.0.2 — Windows

Install

PowerShell irm https://ccisnedev.github.io/finite_ape_machine/install.ps1 | iex

Downloads the latest release, installs to %LOCALAPPDATA%\ape\, adds to PATH, and deploys APE agents to all supported AI coding tools.

Commands

ape target get Deploy agents & skills to all targets
ape target clean Remove deployed files from all targets
ape upgrade Update to the latest release
ape version Print current version
ape init Initialize .ape/ workspace
